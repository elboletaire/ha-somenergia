# Som Energia

Fetch indexed electricity prices and compensation data from Som Energia and expose current/next hour, today, and tomorrow statistics as Home Assistant sensors.

- Select any combination of 2.0TD, 3.0TD, 6.1TD tariffs and compensation during setup.
- Data fetched daily at 18:00 UTC with retries; hourly refresh pushes updated sensor states.
- Minimum Home Assistant version: 2024.7.0.
